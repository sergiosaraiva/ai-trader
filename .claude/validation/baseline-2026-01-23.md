## Architecture

```
Task → Agent Selection
         │
         ├─ Skill Discovery (routing-to-skills)
         │   └─ Score-based selection with confidence thresholds
         │
         ├─ Skill Loading (progressive disclosure)
         │   └─ Core patterns → Examples → Reference files
         │
         └─ Implementation (following skill patterns)
             └─ If skill references another → Skill chaining
```
