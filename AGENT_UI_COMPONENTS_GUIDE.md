## User Interactions

### Starting the Agent
1. User clicks "START AGENT"
2. If mode is "live", confirmation dialog appears
3. On confirm, API call queues start command
4. Button shows "Starting..." with spinner
5. After 2 seconds, status refreshes
6. Component updates to show "RUNNING" state

### Kill Switch
1. User clicks "KILL SWITCH" button
2. Prompt appears: "Enter reason for activation:"
3. On submit, API call triggers kill switch
4. Button shows "Activating..." with spinner
5. After 2 seconds, safety status refreshes
6. All components update to show halted state

### Configuration Changes
1. User clicks gear icon to expand config panel
2. Adjusts sliders/inputs
3. If agent is running, "UPDATE CONFIG" button appears
4. On click, API queues config update command
5. Button shows "Updating..." with spinner
6. Next status poll shows updated config
